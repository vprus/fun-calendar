About

This application simplifies all the complexity of personal scheduling to a single number—the number of hours you have for having fun today. It tracks how many hours your worked this week, remembers about your commute, knows when you need to go to sleep, all so that you can enjoy time away from computers knowing you won't regret it later.

This is HCI class submission. If you are a fellow student, please note that:

  • This is, presently, navigational prototype for assignment 3. Actual functionality to create and edit events, and to update time you have for fun is not implelemented.

  • It requires relatively modern graphical browser, like Chrome, or Firefox. It basically works on iPad, but some transitions are not optimal. It is very explicitly not meant for smartphones.

Today

The today screen shows your agenda for today and how much time you have free. You can click the "Enjoy Now" button when you feel like having fun, and the application will track and update remaining free time.

Week

The week screen shows a week calendar that you can use for a medium term planning. For each week it also shows planned time per category, and how much time do you have to work to meet your weekly goals.
